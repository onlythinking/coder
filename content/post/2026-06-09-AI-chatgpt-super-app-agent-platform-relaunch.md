> 5 天前 ChatGPT 月活刚破 10 亿。OpenAI 转手就要把那个陪你聊天的对话框拆了。

6 月 8 日，英国《金融时报》放出一份十多名 OpenAI 现任和前任员工的集体爆料，揭开了 ChatGPT 自 2022 年上线以来最大规模的产品改版。简单说就是一句话：

**「Chat is dead」—— 聊天框还在，但已经不是产品了。**

## 改版要改什么

接下来几周，ChatGPT 网页端和 App 端会全面改版。干干净净的对话框要消失了。

新版界面里，编程工具、图片生成、第三方合作伙伴的入口会挤进来。Canva 在里面，Booking.com 也在里面。Canva 帮你做海报，Booking.com 帮你订酒店，OpenAI 做入口，第三方做执行。

按 FT 的报道，OpenAI 核心产品负责人 Thibault Sottiaux 的原话是：

> 我们要做的东西将超越现有的界面形态。它会让你拥有一个个人 Agent，能在你生活和工作的方方面面帮到你。手机上、电脑上、浏览器里，甚至在车里，你都能跟它对话。

「你问我答」变成「你说，我去干」。订机票、管日程、写代码、做报表、跑数据分析，ChatGPT 要从聊天对象变成能动手的 Agent。

这条线，OpenAI 在 2025 年埋了整整一年。2025 年 Codex 上线，Agent 时代正式开幕；2026 年 5 月，10 亿月活达成 — 但 OpenAI 内部越来越觉得，这条聊天驱动的增长曲线撑不起一家万亿美元公司的估值。

## 谁在管这件事：Codex 的人接管 ChatGPT

如果说产品改版是表层，组织重组才是真正的信号。

Thibault Sottiaux 在 5 月的组织调整中被提拔到整个核心产品和平台团队的一把手，管辖范围覆盖消费者、企业和开发者三条线。

而从 2022 年起就带着 ChatGPT 一路做到 9 亿周活的 Nick Turley，则被调去了企业产品线。

**换句话说：现在管 ChatGPT 的人是做 Codex 出身的，不是做 ChatGPT 出身的。**

OpenAI 联合创始人 Greg Brockman 在内部备忘录里写的不是「给 ChatGPT 升级」，而是「投资一个统一的 Agent 平台」。几天前的 Intelligence at Work 发布会上，OpenAI 产品负责人更挑明：

> 我们要把 Codex 装进 ChatGPT 里，就这么简单。

ChatGPT、Codex、开发者 API 三条产品线被合并进一个团队。一个不能写代码的 ChatGPT 只是一个聊天界面，一个没有消费者入口的 Codex 只是工程师工具。两者必须合体。

## 超级应用真正争的是什么

当 ChatGPT 帮你订酒店的时候，它调用 Booking 还是 Expedia？当它帮你做海报的时候，它选 Canva 还是 Figma？

**谁来决定这个排序，谁就握着新一代的流量分发权。**

OpenAI 做的不只是 AI 助手，它在抢搜索引擎和应用商店的生意。这不是推测 — 36kr 援引的报道显示，OpenAI 已经开始在 ChatGPT 里向免费用户投放广告，上线六周内年化广告收入突破 1 亿美元。

超级应用的真正价值，不在于功能多，在于占住了用户意图产生的那个瞬间。你刚说出「帮我订明天去上海的机票」，所有的选择权就已经从 Google、从携程、从 Booking 转移到了 ChatGPT 内部的排序算法里。

## Codex 撑得起这个故事吗

Agent 商业化的第一个场景是写代码，原因不复杂：代码有仓库、有测试、有 PR、有 bug、有交付周期，企业能算清楚 ROI。比起「帮我想个营销创意」，Codex 能不能修 bug、跑测试、写内部工具，更容易进入企业预算。

按 36kr 引用的口径，Codex 推出桌面端 App 之后用户量起飞：2 月 80 万，3 月初 160 万，4 月底 400 万，5 月底 500 万 — 半年六倍增长。

更关键的数据是用户构成：目前 Codex 有 20% 的用户不是开发者，知识工作者的增速是开发者的 3 倍以上。OpenAI 也已经为这批用户上线了银行、投研、销售、设计等垂直场景的专用插件。

当一个「编程工具」开始服务银行家和设计师的时候，它就已经不是编程工具了。Codex 正在变成一个覆盖所有知识工作的 Agent 平台。ChatGPT 的那个聊天框，从此只剩下一个功能 — 当入口；后面，跑的全是 Agent。

## 外面追上来的名字

产品要重做，组织要重组，但这一切的一切，指向同一个对手：Anthropic。

5 月 28 日，Anthropic 完成 650 亿美元 H 轮融资，估值 9650 亿，Business Insider 当天发了独家 [Anthropic leapfrogs OpenAI with $965B valuation](https://www.businessinsider.com/anthropic-surpasses-openai-with-965b-valuation-debuts-opus-4-8-2026-5)。这是 Anthropic 第一次在估值上反超 OpenAI。

不过估值只是最表层的压力。**收入端**，Anthropic 的年化营收增速远超 OpenAI；**用户端**，Claude App 月活虽然只有 5600 万（不到 ChatGPT 的零头），但增速是 ChatGPT 的十倍。据报道，美国用户装 Claude App 之后，ChatGPT 的使用时间平均少了 5%。

**编程工具**战场上更直接。截至 2026 年 6 月初，Claude Code 的 GitHub 星标 [131K](https://github.com/anthropics/claude-code)，Codex 仓库 [89.7K](https://github.com/openai/codex)，星标差距比半年前明显收窄。Claude Code 已经在公共 GitHub commit 中占据相当比例（据多家媒体报道在 10% 量级），年初这个数字还是个位数。

> 一年前 OpenAI 的战略是放手一搏，Anthropic 的战略是先把钱赚到手。现在两家在趋同 — 因为都在冲 IPO，投资人关心的是钱，不是梦想。
> —— Jenny Xiao，Leonis Capital 合伙人，前 OpenAI 研究员

## 万亿 IPO 倒计时下的产品抉择

这场趋同的终点在华尔街。5 月 22 日 OpenAI 向 SEC 秘密提交 S-1 招股书，目标估值突破 1 万亿，最早 9 月上市。11 天后 Anthropic 也秘密提交了 S-1，目标 10 月。

但 OpenAI 的财务故事不好讲。月收入约 20 亿美元、年化 250 亿，但每赚 1 美元要亏 1.22 美元。公司自己告诉投资人，不预计 2030 年之前实现正现金流。

「超级应用」是 OpenAI 唯一的路。ChatGPT 作为聊天工具，商业模型太薄 — 10 亿用户里只有 5000 万在付费，付费率不到 5%。剩下 95% 每月白嫖 20 美元 Pro 套餐，靠这个模式给万亿估值讲盈利故事，没有投资人会买账。

但如果 ChatGPT 变成一个 Agent 平台，企业客户为自动化工作流付费，开发者为 API 调用付费，合作伙伴为流量入口付费 — 10 亿用户就变成了全世界最大的销售漏斗。

ChatGPT 名字里的 G 是 Generative、P 是 Pre-trained、T 是 Transformer。但「Chat」这个字母，正在被 OpenAI 自己从产品里抹掉。

## 对开发者的含义

这次改版对开发者意味着几件事：

1. **Codex 的地位会快速上升**。它不再是一个编程工具，而是 OpenAI Agent 平台的载体。Codex CLI、Codex App、Codex API 的整合速度会加快 — FT 报道里 Greg Brockman 的措辞是「投资一个统一的 Agent 平台」，不是「优化开发者体验」。
2. **第三方插件生态会被压缩**。Canva、Booking.com 这样的合作伙伴是新版界面的内容供给方，但分发权完全在 OpenAI 手里。SaaS 公司要面对的事实是：你以为你在接 API，其实你在接受入口排序。
3. **「对话式 UI」正在被重新定义**。对话框不会消失，但它的角色从「产品本身」降级为「指令入口」。真正的产品变成了对话框背后那一组可以调动外部世界（订酒店、写代码、做图、调 API）的 Agent。这件事 Anthropic 的 Claude Code、Anthropic 的 MCP 协议在做，Google 的 Gemini 在做，OpenAI 这次也是明牌跟上。

**AI 产品的终局不是「最好的聊天机器人」，是「最顺手的 Agent 调度器」**。ChatGPT 这三年的演化路径，从插件商店到 GPTs 到 Codex 到超级应用改版，是这个判断的最完整注脚。
