Title: Serena: An open-source MCP toolkit that gives AI coding agents IDE-level code understanding (23k stars)

Body:
Serena (https://github.com/oraios/serena) is an MCP server that provides semantic code retrieval and editing capabilities for AI coding agents.

Key features:
- Symbol-level search (functions, classes, variables) instead of full-text keyword matching
- Cross-file relationship tracking
- 40+ programming languages via LSP
- Integrates with Claude Code, Codex, Cursor, VSCode, JetBrains

As someone who uses AI coding tools daily, this felt like a meaningful improvement over raw file reading.
