**Body**:
Open-Claude-Cowork is a recently released open-source project that wraps Claude's capabilities into a desktop application for developers.

Key features:
- Natural language-driven programming task automation
- File management and batch operations (rename, reorganize, search & replace)
- MCP protocol integration for extended tool ecosystem
- Task chain tracking with version rollback

It's free and open-source, differentiated from GitHub Copilot and Cursor by being fully open with MCP support.
