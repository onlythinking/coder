## Reddit r/programming 草稿

**Title**: Spinel: Matz's experimental AOT compiler for Ruby achieves 11.6x speedup (GitHub, self-hosting)

**Body**:
Matz just published Spinel, an experimental Ahead-of-Time compiler for Ruby. It compiles Ruby source → C code → native binary using standard toolchain (gcc/clang), with whole-program type inference and self-hosting compiler backend.

Key results:
- Conway's Game of Life: 86x faster than CRuby miniruby
- Ackermann: 74x faster
- Mandelbrot: 58x faster
- Self-hosting: compiler written in Ruby, compiles itself to native binary

It's very early (74 tests, 55 benchmarks passing), but the approach is elegant — no custom VM, just Ruby → C → native.

GitHub: https://github.com/matz/spinel
