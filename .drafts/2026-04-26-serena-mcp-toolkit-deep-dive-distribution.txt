**Title**: Serena – A 23K-star MCP-based AI coding toolkit with semantic search and precise editing (deep dive)

**Body**:
I've been exploring AI coding tools and wanted to share a deep dive into Serena, an MCP (Model Context Protocol) based toolkit that recently hit 23K GitHub stars.

**Key points**:
- MCP protocol standardizes how AI tools retrieve project context (code search, file reading, tool invocation)
- Serena provides semantic code search (code embedding-based, not just regex) + AST-aware editing
- Comparison with Continue and Copilot: MCP-native support is the differentiator
- Still early stage: large codebase indexing is slow, MCP ecosystem is evolving

**Link**: https://www.onlythinking.com/post/serena-mcp-toolkit-deep-dive/
