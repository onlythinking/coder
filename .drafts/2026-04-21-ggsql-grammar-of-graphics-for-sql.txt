## Reddit 备选（r/programming）

Title: "ggsql – Posit open-sources a Grammar of Graphics implementation for SQL (alpha)"

Body:
Posit just released ggsql, an alpha-quality tool that lets you describe visualizations directly inside SQL queries using VISUALIZE/DRAW syntax, compiled to Vega-Lite specs.

Key features:
- Declarative GoG syntax in SQL: `VISUALIZE x, y FROM table DRAW point`
- Built-in datasets (penguins, etc.)
- Output: Vega-Lite JSON → renders interactive charts in Quarto, Jupyter, VS Code
- Rust implementation, MIT licensed

Announcement: https://opensource.posit.co/blog/2026-04-20_ggsql_alpha_release/
GitHub: https://github.com/posit-dev/ggsql

For data teams already deep in SQL, this could eliminate the SQL → Python/R → BI tool handoff.
