Title: AI Agent benchmarks are broken — here's why high scores don't translate to real performance

Body:
Most AI coding agents claim impressive scores on standard benchmarks like SWE-Bench, but developers keep finding a huge gap between benchmark performance and real-world usefulness.

This post analyzes the four core problems with current AI agent evaluation:
1. Data contamination — training data "memorizes" benchmark tasks
2. Task granularity mismatch — benchmarks test isolated steps, not multi-round workflows
3. Single-dimensional metrics — only pass/fail, ignoring efficiency and code quality
4. Benchmark pollution — tasks get discussed publicly before evaluation ends

Plus: industry efforts to build better evaluation systems.
