## Reddit 备选（r/programming）

Title: AgentScope Java – An agent-oriented programming framework for building LLM applications in Java (Apache 2.0)

Body:
AgentScope Java (https://github.com/agentscope-ai/agentscope-java) just released version 1.0.11. It's an agent-oriented programming framework specifically for Java developers who want to integrate LLM capabilities into enterprise Java projects.

Key features:
- ReAct (Reasoning + Acting) paradigm with runtime intervention (safe interruption, graceful cancellation, human-in-the-loop hooks)
- Built-in tools: PlanNotebook, Structured Output, Long-term Memory, RAG
- Protocol support: MCP and A2A for tool and agent-to-agent integration
- GraalVM Native Image: 200ms cold start
- OpenTelemetry for distributed tracing
- JDK 17+, Apache 2.0 license

It's built by the same team behind the Python agentscope project, now targeting Java/Spring Boot ecosystem where most existing solutions are Python-first.
