## Reddit (r/programming)

Title:
CrabTrap: Brex open-sourced an HTTP proxy that uses LLM-as-a-judge to secure AI agents in production

Body:
Brex just open sourced CrabTrap, an HTTP/HTTPS forward proxy designed specifically for securing AI agents that make external API calls.

Key features:
- Two-tier decision engine: deterministic static rules + LLM judge
- SSRF protection (blocks RFC 1918 private IPs,169.254.169.254 metadata endpoints)
- Prompt injection defense (JSON encoding/escaping before LLM evaluation)
- Circuit breaker (opens after 5 consecutive LLM failures, retries after 10s)
- Complete audit trail in PostgreSQL

It's MIT licensed and production-ready.

GitHub: https://github.com/brexhq/CrabTrap
Blog: https://www.brex.com/crabtrap

Discussion context: With AI agents increasingly making real API calls (Gmail, Slack, GitHub), security guardrails are critical. CrabTrap provides a clean architectural solution.
