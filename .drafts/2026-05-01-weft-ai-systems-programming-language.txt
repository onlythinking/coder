## Reddit (r/programming) Draft
Title: Weft — a new programming language where AI agents, humans, and APIs are first-class citizens (1160 stars GitHub)

Body:
I came across Weft (https://github.com/WeaveMindAI/weft) today and found the concept interesting — it's a programming language designed specifically for AI system workflows.

Key ideas:
- LLM, Human, HTTP, Database are all built-in node types (no imports needed)
- Fully typed connections between nodes — compiler catches mismatches before runtime
- Programs survive crashes and restarts via Restate (durable execution)
- Two views: code for AI developers, graph visualization for humans — both stay in sync

Example: a human approval workflow that might take 3 days is the same code as one taking 3 seconds.

Seems like an interesting direction for AI workflow orchestration. Would love to hear thoughts from others building similar systems.
